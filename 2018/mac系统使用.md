## 常用的命令
* vim  
```
在命令模式下按i进入输入模式
在输入模式下按ESC切换到命令模式

在命令模式下按:就进入底线命令模式，在底线命令模式下：
q 退出程序
q! 强制离开不存储档案
w 保存文件
wq 存储后离开
wq! 强制存储后离开

```

* 创建文件  
```
touch 文件名称
```

* 按键  
```
home = Fn + 左箭头
end = Fn + 右箭头
PageUp = Fn + 上方向
PageDown = Fn + 下方向
```

* 安装全局npm包没有权限的解决方法。
```
//sudo 表示管理员权限
sudo install package-name -g
```
